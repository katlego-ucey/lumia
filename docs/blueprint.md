# Lumia: Master Blueprint v3.0

## Vision
A 10X-engineered, socially conscious AI Career Agent specifically built for the South African job market.

## Core Pillars
1. **Fraud-protected:** Scanning for job scams (CIPC, domain age, content).
2. **Language-inclusive:** Code-switching support for 11 SA languages.
3. **Education-flexible:** Experience override for non-Matric candidates.

## 6 Functional Loops
1. **Onboarding:** Dialogue-based profile build, language detection, education routing.
2. **CV Architect:** ATS-safe build, no fabrication, multi-profile vault, per-role tailoring.
3. **Interview Mastery:** Scored practice, mastery-gated, Socratic hints, success library.
4. **Job Matching:** 5 elite matches/day, tailored CVs, manual application.
5. **Fraud Guard:** 3 automated gates (CIPC, WHOIS, WhatsApp Sentinel).
6. **Career Navigator:** Gap analysis, milestone roadmap, skill verification.

## 7 AI Agent Roles
1. Dialogue Onboarding Agent
2. Orientation & Training Bootcamp
3. Strategic Career Path Navigator
4. Daily Match & Discovery Engine
5. Profile Vault & CV Morphing Agent
6. Deep Interview Mastery Loop
7. Delayed Retrieval & Remediation Engine

## Technology Stack
- **Orchestration:** LangGraph
- **Models:** LLaMA 3.1 8B (Free), Claude 3.5 Sonnet (Premium)
- **Voice:** Whisper STT, Piper TTS
- **Storage:** PostgreSQL + Redis + pgvector
- **Integration:** Twilio WhatsApp, Adzuna API, SerpAPI, CIPC API, SmileID (POPIA)
- **PDF:** WeasyPrint ATS PDF
